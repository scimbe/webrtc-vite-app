WebRTC Application API

WebRTC Core Components

RTCPeerConnection

Hauptkomponente für die Peer-to-Peer-Verbindung:

  • Verwaltet die Verbindung zwischen Peers
  • Handhabt ICE-Kandidaten und Session-Beschreibungen
  • Koordiniert die Medienübertragung

MediaStream

Verwaltet Audio- und Videostreams:

  • Zugriff auf Kamera und Mikrofon
  • Stream-Verwaltung und Track-Kontrolle
  • Implementiert in useMediaStream Hook

RTCDataChannel

Ermöglicht bidirektionale Datenkommunikation:

  • Peer-to-Peer Datenaustausch
  • Konfigurierbare Übertragungsparameter
  • Ereignisbasierte Kommunikation

React Components

LocalStream Component

Komponente zur Anzeige des lokalen Videostreams

  • Props: Keine erforderlich
  • Funktionen:
    • Automatischer Kamera- und Mikrofonzugriff
    • Fehlerbehandlung bei Medienzugriff
    • Automatisches Cleanup bei Komponenten-Unmount

Custom Hooks

useMediaStream

Hook für Medienzugriff und Stream-Verwaltung

  • Returns:
    • stream: MediaStream | null
    • error: Error | null
  • Features:
    • Automatische Stream-Bereinigung
    • Fehlerbehandlung
    • Status-Management
