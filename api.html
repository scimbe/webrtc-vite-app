WebRTC Application API

WebSocket Integration

WebSocketService

Basis-Service für WebSocket-Kommunikation

  • Methods:
    • connect(): Verbindung herstellen
    • send(message): Nachricht senden
    • onOpen(handler): Event-Handler für Verbindungsaufbau
    • onClose(handler): Event-Handler für Verbindungsabbau
    • onMessage(handler): Event-Handler für eingehende Nachrichten
    • onError(handler): Event-Handler für Fehler
    • disconnect(): Verbindung beenden
  • Features:
    • Automatische Wiederverbindung
    • Nachrichten-Queue
    • Event-Handler-Management

useWebSocket Hook

React Hook für WebSocket-Integration

  • Parameters:
    • url: string - WebSocket-Server-URL
  • Returns:
    • isConnected: boolean - Verbindungsstatus
    • lastMessage: any - Letzte empfangene Nachricht
    • sendMessage: (type: string, payload: any) => void
    • addMessageHandler: (type: string, handler: Function) => () => void

useVideoChat Hook

Hook für Video-Chat-Funktionalität

  • Parameters:
    • roomId: string - Raum-Identifier
  • Returns:
    • isConnected: boolean - Verbindungsstatus
    • peers: Map - Aktive Peer-Verbindungen
    • handleSdpExchange: (peerId: string, sdp: RTCSessionDescription, type: string) => void
    • handleIceCandidate: (peerId: string, candidate: RTCIceCandidate) => void
