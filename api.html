WebRTC Application API

Hooks

usePeerConnection

Hook für WebRTC Peer-to-Peer-Verbindungen

  • Parameters:
    • localStream: MediaStream - Lokaler Audio/Video-Stream
  • Returns:
    • peerConnection: RTCPeerConnection
    • remoteStream: MediaStream
    • connectionState: string
    • createOffer: () => Promise
    • handleAnswer: (answer: RTCSessionDescriptionInit) => Promise
    • handleOffer: (offer: RTCSessionDescriptionInit) => Promise
    • addIceCandidate: (candidate: RTCIceCandidateInit) => Promise
  • Features:
    • Automatische Verwaltung der Peer-Connection
    • ICE-Kandidaten-Handling
    • Stream-Management
    • Verbindungsstatus-Updates
